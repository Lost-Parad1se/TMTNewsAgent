# mp.weixin.qq.com Site Pattern

## 平台特征

- 微信公众号文章常见域名为 `mp.weixin.qq.com`。
- 页面可能依赖完整 URL、query 参数、客户端上下文或用户权限。
- 静态请求可读取部分公开 HTML，但失败、低质量文本或访问提示不等于文章不存在。

## 有效模式

- 优先使用用户提供的完整 URL。
- 不裁剪 query 参数，不重写 `__biz`、`mid`、`idx`、`sn`、`chksm` 等参数。
- 可先尝试单次 static fetch；若用户显式开启本地 CDP，可用本地浏览器辅助读取少量 URL。
- 如果读取失败，进入手动粘贴正文、本地 CSV、授权数据源或第三方合规数据 API 模式。

## 已知陷阱

- 裁剪 query 参数可能导致文章定位失败。
- 静态请求失败、返回短文本或提示“内容不存在”“访问受限”时，不能直接判断文章不存在。
- 本地浏览器读取结果可能受用户登录状态、网络环境和平台策略影响。

## 合规边界

- 不进行高频抓取。
- 不绕过登录、验证码、访问限制、反爬策略或平台权限控制。
- 不操作用户已有浏览器 tab。
- 不保存用户完整浏览器历史。
- 失败时进入手动粘贴正文、CSV 或合规数据源模式。
